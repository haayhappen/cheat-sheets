## Git

### Basics
Init local repo   
```git init <directory> ```

Clone remote repo   
```git clone <repo>```

Stage (specific) files    
```git add <directory>``` or ```git add .``` to add all

Commit with a message   
```git commit -m "<message>"```

Shows staged, unstaged & untracked changes    
```git status```

Display commit history    
```git log```
